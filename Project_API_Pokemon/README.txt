Bienvenido a PokedexPersonal

Gracias por instalar esta aplicación.
Requisitos:
- Windows 10/11 de 64 bits
- Conexión a internet para acceder a la API

Durante la instalación se copiarán los archivos necesarios y se creará un acceso directo en el escritorio.
✅ Cómo se conecta con tu .iss
En tu script de Inno Setup ya puedes poner:

ini
LicenseFile=C:\Proyectos\Project_Manager\Victor_C.V\Project_API_Pokemon\LICENSE.txt
InfoBeforeFile=C:\Proyectos\Project_Manager\Victor_C.V\Project_API_Pokemon\README.txt
De esa forma, el instalador mostrará directamente el contenido de esos archivos en las pantallas de instalación (primero el mensaje de bienvenida, luego la licencia).

👉 Si quieres, te puedo armar un ejemplo de LICENSE.txt y README.txt más detallados, listos para copiar y pegar en tu proyecto. ¿Quieres que te los prepare con un estilo más formal y completo?
