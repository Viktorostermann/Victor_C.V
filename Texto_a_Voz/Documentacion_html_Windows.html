Texto_a_Voz_TkInter_v1 — Documentación (Windows)

Branding: Viktore@ConquerBlocks — Distribución firmada digitalmente.

¿Qué hace este software?

  • Convierte texto a audio.
  • Permite extraer texto desde URLs.
  • Guarda audios en Reportes/Audios_URLS/
  • Guarda logs en Reportes/Logs_GUI/

Requisitos del sistema

  • Windows 10 o Windows 11
  • Arquitectura 64 bits
  • Permisos de instalación (Administrador)

Instalación

Opción 1 — Instalador .exe (Recomendado)

  1. Ejecutar Texto_a_Voz_TkInter_v1_Setup.exe
  2. Seguir el asistente de instalación.
  3. El programa se instalará en: 
    C:\Program Files\Texto_a_Voz_TkInter_v1\

Opción 2 — Desde código fuente

  1. Instalar Python 3.10 (64 bits)
  2. Crear entorno virtual: 
    python -m venv .venv
.venv\Scripts\activate
  3. Instalar dependencias: 
    pip install -r requirements.txt
  4. Ejecutar: 
    python Buscador_Articulo_Texto_Voz.py

Ubicación de datos y reportes

Los reportes se almacenan en:

%APPDATA%\Texto_a_Voz_TkInter_v1\

Firma digital

El ejecutable puede verificarse desde Propiedades → Firmas digitales.

Desinstalación

  • Panel de control → Programas → Desinstalar programa.
  • O desde Configuración → Aplicaciones.

Solución de problemas

  • Si Windows bloquea el ejecutable: clic derecho → Propiedades → Desbloquear.
  • Ejecutar como Administrador si hay problemas de permisos.
